# AIFood 
## 食物辨識熱量管理
### 功能
- 上傳食物照片，辨識食物類別及熱量
- 紀錄本日累積熱量，目標剩餘熱量
- 紀錄體重、BMI
- 個人帳號設定
- 資料視覺化
